Desafíos & Soluciones

⚠ Desafíos Identificados

Sin vista unificada del cliente a través de nafta, tienda y BOXES. Cada canal opera en silos — imposible personalizar ni activar journeys coherentes.

Workflow de aprobaciones SAP atrapado en cadenas de email. Sin contexto enriquecido, múltiples intercambios manuales y riesgo SOX por falta de trazabilidad.

3.000+ técnicos coordinados manualmente, sin inteligencia predictiva. Las violaciones de reglas operativas se detectan tarde — con impacto en seguridad y costos.

Media Cloud licenciado pero sin motor de automatización. Activación de campañas de sponsors y reporting de performance se gestionan manualmente.
✓ Cómo Salesforce lo resuelve

Data Cloud Unified Profile
Data Cloud unifica loyalty, POS, BOXES y digital en un único perfil de cliente para personalización en tiempo real — base del Retail Engagement Agent.

VP Approval Agent (HITL)
Centraliza todas las aprobaciones SAP con contexto enriquecido vía MCP — sin migrar workflows SOX. El humano decide, el agente prepara y registra.

FSM Refinery Agent
Detecta violaciones de reglas en tiempo real (SCADA vía MuleSoft) y optimiza scheduling automáticamente con Tableau Next como superficie de visualización.

Retail Media Agent
Automatiza la activación de campañas para sponsors, reporting de performance y planificación de medios — 100% out-of-box sobre la inversión ya existente.

Flujo Retail Media E2E — Arquitectura Agéntica

Basado en el flujo operativo actual de YPF. Cada paso manual identificado se convierte en una acción agéntica automatizada por Agentforce.

MCI agrega performance de campaña de cada canal
MCI (Marketing Cloud Intelligence) conecta los feeds de performance: Woxi/Intouch (DOOH), Meta, Google, Amazon S3 (data warehouse del sponsor), POS YPF. Unifica impresiones, clics, conversiones y ventas en un único dashboard.
